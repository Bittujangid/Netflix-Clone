# Netflix Clone (HTML & CSS)

A Netflix Clone built using **HTML and CSS**, focused on recreating the Netflix homepage UI with a clean and responsive design.


##  Features

*  Netflix-inspired user interface
*  Responsive design (works on mobile & desktop)
*  Hero section with background
*  Movie/series sections
*  Styled buttons and navigation bar


## Tech Stack

* HTML5
* CSS3


## 📂 Project Structure

```
NETFLIX CLONE
│
├── Material/
│   ├── images/     # Images used in UI
│   ├── videos/     # Video assets
│
├── favicon.ico     # Website icon
├── index.html      # Main HTML file
├── style.css       # Styling file
```

## Purpose

This project was created to:

* Practice HTML & CSS
* Improve layout and design skills
* Understand real-world UI structure
